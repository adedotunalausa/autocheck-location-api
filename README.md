## Description

[Location-Api] is a rest api which collects the information about different locations and also determines the distance between the locations and a supplied base location.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```


## Tools used

```bash
# framework
$ nestjs

# orm
$ typeorm

# database
$ postgres
```
